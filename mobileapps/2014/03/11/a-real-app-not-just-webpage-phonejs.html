A real app, not just a webpage – PhoneJS

You must be wandering why did I wrote like that, well I am not lying, It is truth that using PhoneJS we can build multi-device applications, and I am talking about differnt mobile application platforms, surprised?

PhoneJS is HTML5 JavaScript framework for smartphones and tablets, which contains everything need to create native client applications. It comes with a touch-optimized user experience, natively rendered widgets, straightforward page navigation, view management, and data access layers, PhoneJS is a single page application (SPA) framework for your next iOS, Android and WindowsPhone application, ready for online publication or packaged as a store-ready native app using Apache Cordova (PhoneGap).

It is a single codebase that can be reused across all supported platforms, you can use your skills of HTML5, CSS3 and JavaScript to build such applictions. PhoneJS libraries fully leverage jQuery and optionally support Knockout for MVVM user interface development. In general there is no need to learn a single native platform SDK to build application, when you have opt for PhoneJS.

Development approaches like jQuery and Knockout are choices for developer to pick either of them for development.

Features list: * SPA View Management * URL Routing * Flexible View Rendering Engine * Partial Views * Device-dependent Views * Master Pages or Layouts * Pluggable template engine * Separate view files * Adaptive Layout System * Navbar Layout * SlideOut Layout * View Caching and State Management

For more details and in depth knowledge of features please visit http://phonejs.devexpress.com
